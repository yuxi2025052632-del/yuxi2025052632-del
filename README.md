## 关于我

我主要关注自动化、Bot 系统、Dashboard 工具以及日常运维流程优化。  
相比重复性的手工操作，我更倾向于通过脚本、命令体系和可复用工作流，把问题稳定地工程化解决。

目前长期在做的方向包括：
- 基于命令的 Bot 功能设计与扩展
- 运维后台 / Dashboard 的功能开发与交互改进
- 图像卡片、渲染器与数据处理流程
- 用 Python / Node.js 编写实用型工具与自动化脚本
