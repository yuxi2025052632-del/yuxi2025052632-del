## About

主要做自动化、Bot 工程、Dashboard 工具以及运维相关的效率优化。  
相比重复性的手工操作，我更倾向于通过脚本、命令体系和可复用工作流，将问题工程化、工具化、流程化地解决。

长期关注的方向包括：
- 基于命令的 Bot 功能设计与扩展
- 运维后台 / Dashboard 功能开发与交互优化
- 图片卡片、渲染器与数据处理流程
- Python / Node.js 实用工具与自动化脚本
- 面向日常场景的小型高频效率工具
